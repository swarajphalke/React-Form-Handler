## 🚀 Features

- **Form Handling:** Input capture, validation, and submission.
- **Comment Form:** Simple comment input component.
- **Counter Component:** Interactive counter using React state.
- **Modular Components:** Organized and reusable React components.
- **Vite:** Blazing fast dev server and build tool.
- **Styling:** Custom styles using CSS.

## 📦 How to Use

1. Clone the repository:
```bash
git clone https://github.com/swarajphalke/note-taker.git
```

2. Navigate to the project directory:
```bash
cd React-Form-Handler
```

3. Install dependencies:
```bash
npm install
```

4. Run the development server:
```bash
npm run dev
```

5. Builds the app for production:
```bash
npm run build
```
